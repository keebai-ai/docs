## ¿Qué son los Tags?

Los tags (etiquetas) son una forma rápida de clasificar y agrupar contactos. Puedes usar tags para segmentar tu base, filtrar contactos y seleccionar destinatarios en [difusiones masivas](/canales/difusion-masiva).

## Crear un tag

<Steps>
  <Step title="Ve a Tags">
    Navega a **CRM → Tags** y haz clic en **+ Nuevo Tag**.
  </Step>
  <Step title="Define el nombre y color">
    Elige un nombre descriptivo (ej: "VIP", "Lead caliente", "Soporte resuelto") y un color para identificarlo visualmente.
  </Step>
</Steps>

## Asignar tags a contactos

Puedes asignar tags de varias formas:

| Método | Cómo |
|--------|------|
| **Desde el perfil** | Abre un [cliente](/crm/clientes) y haz clic en **+ Tag** |
| **Desde la conversación** | En la ventana de chat, usa el botón de etiquetas |
| **Masivamente** | Selecciona varios contactos en la lista y asigna un tag a todos |
| **Automáticamente** | Los [Flows](/canales/flows) pueden asignar tags como parte de un flujo |
| **Por el asistente de IA** | Configura tu [asistente](/ia-asistentes/mis-asistentes) para que etiquete contactos según la conversación |

## Usos comunes de tags

<CardGroup cols={2}>
  <Card title="Segmentar audiencias" icon="users">
    Crea tags como "Interesados en Producto A" para luego enviarles una difusión específica.
  </Card>
  <Card title="Marcar estado" icon="flag">
    Usa tags como "Pendiente de respuesta", "Documentación enviada" o "Cerrado" para rastrear avances.
  </Card>
  <Card title="Clasificar origen" icon="location-dot">
    Tags como "Evento CDMX", "Referido" o "Ads Facebook" para saber de dónde viene cada contacto.
  </Card>
  <Card title="Priorizar" icon="star">
    Tags como "VIP" o "Lead caliente" para que tu equipo sepa a quién atender primero.
  </Card>
</CardGroup>

<Tip>
  Mantén tus tags organizados. Revisa periódicamente y elimina los que ya no uses para evitar desorden.
</Tip>
