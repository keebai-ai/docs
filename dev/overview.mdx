La API pública de Keebai te permite integrar tu CRM, tu propio backend, automatizaciones (Zapier, n8n, Make) y cualquier herramienta interna con tus canales de WhatsApp en Keebai. Está pensada para desarrolladores y administradores con perfil técnico.

## Características

<CardGroup cols={2}>
  <Card title="REST + JSON" icon="code">
    Todos los endpoints aceptan y devuelven JSON. Versionados bajo `/v1/`, sin breaking changes silenciosos.
  </Card>
  <Card title="Personal Access Tokens" icon="key">
    Cada usuario del tenant puede generar sus propios tokens, con scopes acotados y revocación inmediata.
  </Card>
  <Card title="Scopes granulares" icon="shield">
    Distintos permisos para envío individual, envío masivo y lectura de templates. Asigná solo lo necesario por integración.
  </Card>
  <Card title="OpenAPI 3" icon="file-code">
    Especificación auto-generada en [api.keebai.com/v1/docs-json](https://api.keebai.com/v1/docs-json). Importable en Postman, Insomnia o cualquier generador de SDK.
  </Card>
</CardGroup>

## Base URL

```
https://api.keebai.com/v1
```

Todas las rutas de la API pública viven bajo este prefijo. La interfaz de Swagger UI navegable está en [api.keebai.com/v1/docs](https://api.keebai.com/v1/docs).

## Endpoints disponibles

| Método | Ruta | Scope requerido | Descripción |
|---|---|---|---|
| `GET` | `/v1/me` | — | Información del token autenticado y sus scopes. |
| `POST` | `/v1/messages/template` | `messages:send` | Envía un template aprobado a un destinatario. |
| `POST` | `/v1/messages/text` | `messages:send` | Envía texto libre dentro de la ventana de 24h. |
| `POST` | `/v1/messages/bulk` | `messages:bulk` | Envía un template a múltiples destinatarios (hasta 5000). |
| `GET` | `/v1/messages/bulk/:broadcastId` | `messages:bulk` | Estado de un envío masivo. |
| `GET` | `/v1/templates` | `templates:read` | Lista templates aprobados de la company. |

## Roadmap

Próximas iteraciones planificadas, sin fecha comprometida:

- **Smart templates**: mapeo de datos no estructurados a variables de template usando LLM.
- **Mensajes agendados**: agendamiento y cancelación de envíos a futuro.
- **Tickets**: apertura y actualización de tickets desde la API.
- **Webhooks**: notificaciones push para eventos de mensajes (delivered, read, failed).

Si necesitás algo de esto antes que el resto, escribinos a [support@keebai.com](mailto:support@keebai.com).
