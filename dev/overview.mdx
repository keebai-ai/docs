La API pública de Keebai te permite integrar tu CRM, tu propio backend, automatizaciones (Zapier, n8n, Make) y cualquier herramienta interna con los datos de tu cuenta. Está pensada para desarrolladores y administradores con perfil técnico.

## Características

<CardGroup cols={2}>
  <Card title="REST + JSON" icon="code">
    Todos los endpoints aceptan y devuelven JSON. Versionados bajo `/v1/`, sin breaking changes silenciosos.
  </Card>
  <Card title="Personal Access Tokens" icon="key">
    Cada usuario del tenant puede generar sus propios tokens, con scopes acotados y revocación inmediata.
  </Card>
  <Card title="Menor privilegio" icon="shield">
    Los scopes del token son siempre un subconjunto de los permisos del usuario. Si el token leakea, el daño queda contenido.
  </Card>
  <Card title="OpenAPI 3" icon="file-code">
    Especificación auto-generada en [api.keebai.com/v1/docs-json](https://api.keebai.com/v1/docs-json). Importable en Postman, Insomnia o cualquier generador de SDK.
  </Card>
</CardGroup>

## Base URL

```
https://api.keebai.com/v1
```

Todas las rutas de la API pública viven bajo este prefijo. La interfaz de Swagger UI navegable está en [api.keebai.com/v1/docs](https://api.keebai.com/v1/docs).

## Disponibilidad

La API pública se está liberando endpoint a endpoint. Hoy está disponible:

- `GET /v1/me` — información del token autenticado y permisos efectivos.

Próximamente: contacts, conversations, broadcast, CRM, webhooks. Si necesitás un endpoint específico antes que el resto, escribinos a [support@keebai.com](mailto:support@keebai.com).
