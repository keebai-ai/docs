Devuelve la cita indicada siempre que pertenezca al tenant del PAT.

## Endpoint

```
GET https://api.keebai.com/v1/scheduling/appointments/:id
```

## Scope requerido

`scheduling:appointments:read`

## Path params

| Param | Tipo | Descripción |
|---|---|---|
| `id` | `string` | Id de la cita (UUID generado al crearla). |

## Ejemplo de request

```bash
curl https://api.keebai.com/v1/scheduling/appointments/9d5b1f4e-7c2a-4a4d-9d3e-9f1b1f8c7a3a \
  -H "Authorization: Bearer kbai_pk_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"
```

Mismas semánticas que el resto de la API pública.
