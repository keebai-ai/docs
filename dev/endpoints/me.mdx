Devuelve los scopes del token autenticado tal como están guardados en la base. Sirve principalmente para:

- Validar al inicio de tu integración que el token está vivo y tiene los scopes que esperás.
- Mostrar al usuario final (en una UI de configuración de tu app) qué acciones va a poder hacer.
- Detectar si el token fue revocado (el endpoint devolverá `401`).

## Endpoint

```
GET https://api.keebai.com/v1/me
```

## Headers

| Header | Requerido | Valor |
|---|---|---|
| `Authorization` | Sí | `Bearer kbai_pk_<token>` |

No requiere body, query params ni path params.

## Ejemplo de request

<CodeGroup>

```bash curl
curl https://api.keebai.com/v1/me \
  -H "Authorization: Bearer kbai_pk_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"
```

```js JavaScript
const response = await fetch("https://api.keebai.com/v1/me", {
  headers: { Authorization: `Bearer ${process.env.KEEBAI_API_TOKEN}` },
});
const { permissions } = await response.json();
```

```python Python
import os, requests

resp = requests.get(
    "https://api.keebai.com/v1/me",
    headers={"Authorization": f"Bearer {os.environ['KEEBAI_API_TOKEN']}"},
    timeout=10,
)
resp.raise_for_status()
permissions = resp.json()["permissions"]
```

</CodeGroup>

## Respuesta

### 200 OK

```json
{
  "permissions": [
    "messages:send",
    "templates:read"
  ]
}
```

| Campo | Tipo | Descripción |
|---|---|---|
| `permissions` | `string[]` | Scopes del token. Equivale al array `scopes` que se eligió al crearlo. No varía entre requests salvo que generes un token nuevo. |

### 401 Unauthorized

Token inválido, revocado o expirado. Ver [Errores de autenticación](/dev/authentication#errores-de-autenticación).

```json
{
  "message": "Invalid or missing API token",
  "error": "Unauthorized",
  "statusCode": 401
}
```

### 429 Too Many Requests

Pasaste el rate limit. Ver [Rate limits](/dev/rate-limits).

## Patrones comunes

### Validar al inicio de la integración

```python
import os, requests, sys

REQUIRED = {"messages:send", "templates:read"}

resp = requests.get(
    "https://api.keebai.com/v1/me",
    headers={"Authorization": f"Bearer {os.environ['KEEBAI_API_TOKEN']}"},
    timeout=10,
)
if resp.status_code == 401:
    sys.exit("Token inválido o revocado")
resp.raise_for_status()

scopes = set(resp.json()["permissions"])
missing = REQUIRED - scopes
if missing:
    sys.exit(f"Faltan scopes: {sorted(missing)}")

print("Token OK con scopes:", sorted(scopes))
```

### Health check de tu integración

Llamar `GET /v1/me` cada 5 minutos como liveness probe interno: si recibís `401` reiteradamente, alertás a tu equipo de que el token fue revocado o expiró.
