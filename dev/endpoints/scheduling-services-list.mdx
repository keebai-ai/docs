Devuelve los servicios configurados en el tenant resuelto desde el PAT. La búsqueda usa el índice de texto del servicio (`name`, `description`) en español. Los servicios incluyen duración, restricciones horarias y profesionales/recursos permitidos para agendar.

## Endpoint

```
GET https://api.keebai.com/v1/scheduling/services
```

## Scope requerido

`scheduling:services:read`

## Query params

| Param | Tipo | Default | Descripción |
|---|---|---|---|
| `branchId` | `string` | — | Filtrar servicios disponibles en una sucursal específica. |
| `search` | `string` | — | Búsqueda full-text sobre nombre y descripción. |
| `status` | `string` | — | Filtrar por estado del servicio (`active`, `inactive`). |
| `skip` | `number` | `0` | Offset de paginación. |
| `limit` | `number` | `50` | Tamaño de página (máximo 200). |

## Respuesta

### 200 OK

Lista de servicios. Cada item incluye al menos:

| Campo | Tipo | Descripción |
|---|---|---|
| `_id` | `string` | Id del servicio. Se usa como `service_id` en disponibilidad y al crear citas. |
| `name` | `string` | Nombre visible del servicio. |
| `description` | `string` | Descripción libre. |
| `duration_minutes` | `number` | Duración por defecto en minutos. Define el tamaño del slot. |
| `bufferBefore` / `bufferAfter` | `number` | Minutos de buffer alrededor del slot. |
| `time_rules` | `object` | `{ earliest_start, latest_end }` — restringe la franja horaria en la que se puede agendar. |
| `allowedProfessionals` | `string[]` | Ids de profesionales habilitados para prestar el servicio. |
| `price` / `currency` | — | Tarifa de referencia. |
| `status` | `string` | `active` o `inactive`. |

### 401 Unauthorized · 403 Forbidden

Mismas semánticas que el resto de la API pública.
