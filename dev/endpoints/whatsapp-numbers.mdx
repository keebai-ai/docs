Devuelve los números WhatsApp Business conectados a la company, en un shape liviano. Es una proyección de [GET /channels?type=whatsapp](/dev/endpoints/channels-list) optimizada para los flujos donde solo necesitás identificadores Meta.

```http
GET /v1/whatsapp/numbers
```

**Scope requerido:** `channels:read`

## Request

```bash
curl https://api.keebai.com/v1/whatsapp/numbers \
  -H "Authorization: Bearer kbai_pk_xxx"
```

## Response 200

```json
{
  "data": [
    {
      "id": "1234567890",
      "phone_number": "+56987654321",
      "waba_id": "9876543210",
      "channel_id": "ch_01HXYZ...",
      "status": "active"
    }
  ]
}
```

| Campo | Tipo | Descripción |
|---|---|---|
| `id` | string | `phone_number_id` de Meta. Usalo en `POST /v1/messages` (`from_phone_number_id`) cuando tengas más de un número. |
| `phone_number` | string | Número en formato E.164. |
| `waba_id` | string | WhatsApp Business Account id de Meta. |
| `channel_id` | string | Channel id de Keebai. Mismo que aparece en `/v1/channels`. |
| `status` | enum | `active` \| `paused` \| `disconnected`. |
