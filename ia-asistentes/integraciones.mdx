Las integraciones permiten que tus asistentes se comuniquen con herramientas externas: consultar APIs, sincronizar datos y ejecutar acciones fuera de Keebai.

## Tipos de integración

<CardGroup cols={2}>
  <Card title="APIs externas" icon="code">
    Conecta con tu propio backend o cualquier API para consultas o acciones específicas.
  </Card>
  <Card title="Webhooks" icon="globe">
    Envía datos a cualquier URL cuando ocurra un evento (nuevo lead, conversación cerrada, etc.).
  </Card>
  <Card title="Sincronización de datos" icon="arrows-rotate">
    Mantén información sincronizada entre Keebai y tus sistemas externos.
  </Card>
  <Card title="Automatización" icon="bolt">
    Dispara acciones automáticas en herramientas externas desde una conversación.
  </Card>
</CardGroup>

## Conectar una integración

<Steps>
  <Step title="Ve a Integraciones">
    Navega a **IA & Asistentes → Integraciones**.
  </Step>
  <Step title="Selecciona o crea una integración">
    Configura la conexión: URL del endpoint, método HTTP, headers de autenticación y parámetros.
  </Step>
  <Step title="Prueba la conexión">
    Usa el botón de prueba para verificar que la integración responde correctamente.
  </Step>
  <Step title="Vincula a una función">
    Usa la integración como acción dentro de una [función](/ia-asistentes/mis-funciones) para que tu asistente pueda invocarla durante una conversación.
  </Step>
</Steps>

## Ejemplos de uso

| Integración | Lo que hace |
|-------------|------------|
| Consultar disponibilidad en Google Calendar | El asistente revisa horarios libres y los ofrece al cliente |
| Registrar lead en un CRM externo | Al captar datos del cliente, se envían automáticamente |
| Webhook a Slack | Notifica a tu equipo cuando ocurre un evento importante |
| API de inventario | El asistente consulta stock en tiempo real |

<Warning>
  Las integraciones con APIs externas pueden requerir configuración técnica (API keys, OAuth, etc.). Si necesitas ayuda, contacta a [support@keebai.com](mailto:support@keebai.com).
</Warning>
